# Arabic UI Glossary (Dashboard)

## Core Terms
- Advisor → المرشد
- Advisor ID → معرّف المرشد
- Student → الطالب
- Course → المقرر
- Section → الشعبة
- Program → البرنامج
- Department(s) → القسم / الأقسام
- Role → الدور
- Scope → النطاق

## Actions
- Load → تحميل
- Run → تشغيل
- Check → فحص
- Save → حفظ
- Apply → تطبيق
- Copy → نسخ
- Export → تصدير
- Import → استيراد
- Delete → حذف
- Preview → معاينة
- Refresh → تحديث
- Enable → تفعيل
- Disable → تعطيل

## Reporting / Ops
- Debug Report → تقرير التصحيح
- Eligibility → الأهلية
- Eligibility Inspector → فاحص الأهلية
- Batch Recommender → توصية الدفعة
- Student Recommender → توصية الطالب
- Mapping → الربط
- Filters → المرشحات
- High Priority → أولوية عالية
- Missing High Priority → نواقص عالية الأولوية
- Integrity Report → تقرير السلامة

## UI Phrases
- No data yet. → لا توجد بيانات بعد.
- No action yet. → لا يوجد إجراء بعد.
- Network request failed. → فشل طلب الشبكة.
- Failed to load ... → فشل تحميل ...
- Loaded successfully. → تم التحميل بنجاح.

## Style Conventions
- Use clear Modern Standard Arabic, simple and operational.
- Prefer short actionable labels on buttons.
- Keep technical tokens unchanged when needed (CSV, XLSX, IDs, code fields).
- Use imperative form for actions (تشغيل، تحميل، تطبيق).
